Module ListDir by Tricky

Functions Summary

ListDir Returns a TList containing all files inside a dir. It does not work recursively (for that use the Tree module). If the t parameter is set to LISTDIR_FILEONLY then you'll only get files and if it's set to LISTDIR_DIRONLY it will only contain directories (or folders if you like). When set to 0 it will add directories and folders alike.
