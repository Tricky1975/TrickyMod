Function FuncPicker:String(Lua:Object,Caption$="Please pick a function:")
DescriptionScans a Lua file for functions, and let's the user pick one and returns that as a string.
InformationThe 'Lua' variable can be either a filename or it can be a memory bank in which the code is loaded (by means of 'LoadBank' for example, but also handy to use with JCR5's JCR_B() function.

If you want to read the stuff directly from a string containing the script, use the FuncFromScriptPicker function in stead!
