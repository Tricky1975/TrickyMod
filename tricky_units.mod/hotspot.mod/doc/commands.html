Module HotSpot by Tricky

Functions Summary

HotCenter Move the hotspot to the center of the picture.
HotEnd Move the hotspot to the bottom right of the picture.
HotSpot Set the Hotspot (or image handle) to a basic spot.

Functions

Function HotCenter(I:TImage)
DescriptionMove the hotspot to the center of the picture.

Function HotEnd(I:TImage)
DescriptionMove the hotspot to the bottom right of the picture.

Function HotSpot(I:TImage,HX,HY)
DescriptionSet the Hotspot (or image handle) to a basic spot.
InformationIn stead of coordinates (as we already got SetImageHandle for that) this routine just goes for HS_START, HS_CENTER and HS_END. The computer will calculate the actual coordinates automatically.
