Function CreateTree:TList(Dir$="",Sort=False,allowhidden=False)
DescriptionAnalysis the directory tree of a folder and stores all files in a TList containing all file names in strings.
InformationThis function just adds the files as they pass through. When you set the 'Sort' parameter, all files will be put into alphabetic order.
