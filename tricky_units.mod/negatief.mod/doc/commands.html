Module Negatief by Tricky

Functions Summary

ImgNegatief Inverts an image coloring.
PixNegatief Inverts a pixmap coloring in order to create a "negative".

Functions

Function ImgNegatief:TImage(Image:Object)
ReturnsThe inverted image as a TImage.
DescriptionInverts an image coloring.
InformationThis function has been setup to support both single pic images as animated images (though I must admit that animated images do not work the way they should yet, and I don't know what causes it. If you can fix it, let me know).
The image can be an existing image, a Pixmap, a file name, or a bank containing the contents of a picture file (provided the loader for it is imported in your program).

Function PixNegatief:TPixmap(Pixmap:TPixmap)
ReturnsThe Pixmap containing the reverted image.
DescriptionInverts a pixmap coloring in order to create a "negative".
