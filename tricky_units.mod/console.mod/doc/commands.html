Globals

Global ConsoleBackGroundPicture:TImage
DescriptionContains the image used for the background of the console.
InformationJust assign and image to it and the console will tile it onto the background. Animation not yet supported, maybe in future versions!

Global ConsoleCommand$
DescriptionWhen the user types a command into the console, the result will be stored into this variable.

Functions

Function ConsoleCloseLogFile()
DescriptionWill close the Console Log file.
InformationFor proper usage it's always advisable to do this before ending your program.

Function ConsoleDoublePrint(T1$,T2$,R1=255,G1=255,B1=255,R2=255,G2=255,B2=255)
DescriptionPuts first text at the start and the second text halfaway the screen. Handy for some upsummings.

Function ConsoleDoublePrint2(T1$,T2$,Offset,R1=255,G1=255,B1=255,R2=255,G2=255,B2=255)
DescriptionSame as ConsoleDoublePrint but the offset allows you the define X coordinate on where to start the second collumn.

Function ConsoleFont:TImageFont()
DescriptionContains the font that the console uses.
InformationYou can also load it by reading it from 'incbin:unispace.ttf', but be warned... You never know if I ever pick a different font for this so using this function is a safer route.

Function ConsoleModify(AConsoleLine:Object,Txt$,R=-1,G=-1,B=-1)
DescriptionAllows you to modify a line present on the console.
InformationIn order to modify you must have stored the return value of ConsolePrint into a variable. Next make sure the line you want to modify is still on the screen or else nothing will happen.

Another note... When you created a log file it will not be altered by this routine!


Function ConsolePrint:ConsoleLine(Txt$,R=255,G=255,B=255)
DescriptionPuts text into a new console line.

Function ConsoleShow()
DescriptionShow the contents of the console onto the screen.
InformationOf course you need to use the Flip command before you can actually see it ;)

Function ConsoleWrite:ConsoleLine(Txt$,R=255,G=255,B=255)
DescriptionAlias for ConsolePrint()

Function ConsoleWriteLogFile(LogFile$,BGCOLOR$="#000000")
DescriptionWill start a log file that logs all stuff that is put onto the console. The output is in HTML code.
InformationThough the result of this file should be readable by any browser I only tested this out in Firefox and Apple Safari. If there are issues with Opera and Chrome I'm willing to look into that but I will under NO CONDITION work this out in IE (because a. that browser was screwed from it's very first release, and b. since my main system is not Windows, it's too much trouble to test this properly).
