DescriptionIf the last input request was actually accepted then this value will contain the value 'True', in any other case this value contains the value 'False'

Function MaxGUI_Input$(Caption$="Please enter a character string:",DefaultText$="",AllowCancel=True,AllowedChars$="")
DescriptionPuts up a quick diaglog box and returns a string of what the user wanted.
InformationMost parameters are self-explaining. The 'AllowedChars' one will only allow those characters in the text you have chosen. When you leave it empty all characters the user can enter are allowed.
