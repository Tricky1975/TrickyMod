Module jcr6.jcr5driver

This module is a driver to read JCR5 files in JCR6. All you need to do is to import this module and your set.

Please note that JCR5 was decprecated from the day that JCR6 was released. This module only serves for backward compatibility purposes. JCR5 will no longer be maintained, unless somebody wishes to take that one over from me.
