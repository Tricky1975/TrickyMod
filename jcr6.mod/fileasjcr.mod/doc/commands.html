Module jcr6.fileasjcr

This module has a quick routine to load raw files as a one-entry JCR file. This routine was most of all written with patching incbin files in mind, allowing a handy provision of built in files in engines, aside from the extra assets.

Functions Summary

Raw2JCR Loads a raw file as it were a one-entry JCR file. Handy For quick patching raw files into a great JCR file.
