GALE.M2D: Modinfo Source  

This file contains a driver to make GALE operate from a Max2D environment.

Please note that not all of Gale's sub modules will operate properly from this environment.

NEVER import this module together with GALE.MGUI or your project will cause conflicts which will lead to very terrible results in the worst case even system crashes.
